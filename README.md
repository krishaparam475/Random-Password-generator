# OIBSIP_python_taskno1
Objective:

It is a command-line password generator in Python that generates random passwords based on user-defined criteria, such as length and character types (letters, numbers, symbols). Allow users to specify password length and character set preferences.

Steps Performed:

1.  Imported the required random and string modules.
   
  2. Collected user inputs for password length and character preferences.
   
  3. Combined the selected character sets (letters, numbers, and symbols).
   
  4. Generated a random password using the chosen criteria.
   
  5. Displayed the generated password and handled invalid input when no character type was selected.

Tools Used:

Programming Language: Python

IDE: Visual Studio Code

Python Libraries:
        random
        string
    
Outcome

Successfully created a command-line application that generates strong, random passwords based on user preferences. The program is simple, efficient, and demonstrates the use of Python's built-in libraries for randomization and string manipulation.
